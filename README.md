# @raae/gatsby-plugin-let-it-snow

> Amazing plugin that solves everything

## How to install

`npm install @raae/gatsby-plugin-let-it-snow`

or

`yarn add @raae/gatsby-plugin-let-it-snow`

## How to use

```
module.exports = {
  plugins: [
    `@raae/gatsby-plugin-let-it-snow`
  ],
}
```
